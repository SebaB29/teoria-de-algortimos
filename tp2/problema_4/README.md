# Trabajo Práctico 2 - Problema 4

## Requisitos

- Python 3.10 o superior
- Bibliotecas: `matplotlib`

## Estructura

- `problema_4.py`: implementación principal
- `generar_metricas.py`: script para calcular métricas y producir `promedios_obtenidos.csv` y figuras (`grafico_promedios.png`).
- `promedios_obtenidos.csv`: salida de métricas agregadas.

## Ejecución

Generar métricas y gráficos:

```bash
python3 generar_metricas.py
```

## Salida

- `promedios_obtenidos.csv`: contiene los promedios calculados por `generar_metricas.py`.
- `grafico_promedios.png`: gráfico generado con los promedios.
